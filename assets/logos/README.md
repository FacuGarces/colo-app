# Logos de las listas

Estos son **placeholders** generados con el número y color de cada lista.
Para reemplazar por los logos oficiales:

1. Conseguí el logo oficial (idealmente SVG o PNG con fondo transparente, cuadrado).
2. Guardalo con el nombre correspondiente (`lista4.svg`, `lista9.svg`, etc.) sobrescribiendo el placeholder.
3. Si usás PNG en vez de SVG, cambiá también el path en `js/config.js` (campo `logo`).

## Correspondencia
- `lista4.svg` → En Clave Roja
- `lista8.svg` → El Frente
- `lista9.svg` → **Somos Libres** (la nuestra)
- `lista10.svg` → Nuevo Espacio
- `lista12.svg` → El Torrente
- `lista13.svg` → Nuevo MAS
- `lista17.svg` → JUP
- `listax.svg` → Lista peronista por confirmar
